# 周报生成技能

## 技能说明

本技能包含：
- **方法论指导**：告诉 Agent 如何从工作记录中提炼周报内容
- **可执行工具**：`tools.py` 提供 `generate_report()` 函数，将周报内容渲染为 HTML 文件

## 使用流程

1. 回顾这段时间的操作记录（shell 命令、文件改动等）
2. 按分类整理：新功能、问题修复、优化、其他
3. 每个类别下列出具体事项，附上关键数据
4. 调用 `generate_report()` 生成 HTML 周报文件

## 输出要求

周报内容需包含：
- 本周工作概览（一句话总结）
- 分类详情（每类至少一条）
- 待办事项（下周计划）
